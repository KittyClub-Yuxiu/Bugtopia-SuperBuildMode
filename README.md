**超级建造模式 (Super Build Mode)**

**简介：**
你是否也曾为装饰品之间那恼人的碰撞体积而烦恼？你是否也曾想将一株植物悬浮在空中，却被无情的地面所束缚？

现在，“超级建造模式” 将解放你的全部创造力！这个Mod移除了游戏中绝大部分的建造限制，让你能够以前所未有的自由度，随心所欲地打造专属于你的、独一无二的虫虫生态箱。

**核心功能：**
✅ 无视重叠： 现在所有造景物都不再有重叠检测，你可以随机放置造景物，创造出更复杂、更精细的景观。告别“红色”警告！

✅ 自由悬空： 不再需要地面支撑！让你的植物、岩石和所有装饰品，都按照你的意志飘浮在空中。

✅ 正常保存： 你的所有“不合规则”的艺术品都可以被完美地保存和加载，无需担心存档损坏。

**【安装指南】**
安装 BepInEx： 如果你是第一次使用Mod，请先从 [BepInEx的GitHub发布页](https://github.com/BepInEx/BepInEx/releases) 下载最新版本的 BepInEx。

解压： 将压缩包内的所有文件，解压到《虫虫生态箱》的游戏根目录 (也就是与 Bugtopia.exe 文件放在一起)。

安装本Mod： [下载 SuperBuildMode.dll](https://github.com/KittyClub-Yuxiu/Bugtopia-SuperBuildMode/releases)。

放入插件文件夹： 将 SuperBuildMode.dll 文件，放入游戏根目录下的 BepInEx/plugins 文件夹内。(如果 plugins 文件夹不存在，请先运行一次游戏，它会自动生成)。

完成！启动游戏即可享受！

**【使用方法】**
本Mod无需任何额外操作。安装成功后，只需进入游戏的“造景模式”，你就会发现那些烦人的限制已经消失了！
